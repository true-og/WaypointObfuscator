# WaypointObfuscator

Requires: https://github.com/retrooper/packetevents

WaypointObfuscator prevents malicious client mods from discovering which waypoint belongs to which player by obfuscating the waypoint identifier uniquely for each player.

This doesn’t solve the problem of players using triangulation to find others coordinates, but it still makes it harder to determine which waypoint belongs to whom.
